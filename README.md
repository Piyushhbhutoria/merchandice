# Merchandice (GitHub Pages)

This repo is a Jekyll site meant for GitHub Pages.

## GitHub Pages Setup

1. Push this repo to GitHub.
2. In the repo settings, go to **Pages**.
3. Choose **Deploy from a branch**, select the default branch and `/ (root)` folder.
4. Save and wait for the Pages build to complete.

If using a custom domain, add a `CNAME` file at the repo root.
